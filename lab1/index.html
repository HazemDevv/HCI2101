Introduction to 3D Modelling and Animation

Course Code: HCI2101

This course provides an in-depth introduction to 3D modelling and animation, equipping students with the technical and creative skills necessary to design and animate 3D assets.

Course Learning Outcomes (CLOs)

Code Description
K1 Understand the principles of 3D design and texturing.
S1 Use standard tools such as Autodesk Maya and Blender.

Course Topics

  1. Introduction to 3D Space and Navigation
  2. Polygon Modeling Techniques
  3. Texturing and Materials application
  4. Lighting and Rendering basics
  5. Principles of Animation and Rigging

Student Assessment

Assessment Task Percentage
Midterm Exam 30%
Practical Project 30%
Final Exam 40%
